# HTML Enter++

## Features
HTML Enter++ adds keybindings for `Enter` while editing HTML files.

## Default Key bindings
|||
|---:|----|
|`Shift` + `Enter`|Next Line (insert "**\<br\>** + \\n")|
|`Ctrl(Cmd)` + `Enter`|Next Paragraph (insert "\\n + **\</p\>** + \\n + **\<p\>** + \\n")|
